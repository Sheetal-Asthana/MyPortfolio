WORD VOYAGE

A fun and interactive quiz application that challenges users with dynamic questions, real-time scoring, and engaging visuals. Designed for both learning and entertainment, this app makes testing knowledge an exciting experience!

Home Page

Displays the app's logo and name.

Offers options to start a quiz, write your name, or exit the quiz.





Fill in your name and click on the start button to start the quiz.





Quiz Instructions Page

Provides an overview of the quiz rules (e.g., number of questions, time limit, scoring system).

Includes tips or guidelines for playing.





Quiz Question Page

Displays one question at a time.

Includes multiple-choice answers, true/false options.

Has a countdown timer for time-limited quizzes.

Has a 50-50 Lifeline option that can only ne used once throughout the quiz.

Shows navigation buttons (Next, Previous, Submit).





Quiz Results Page

Displays the user's score and performance analysis.

Provides options to retry the quiz with play again button.
